## 11. Emergency Protocol Flowchart

```
INCIDENT OCCURS (medical, missed ferry, lost document, etc.)
                │
                ▼
Is the pet in immediate medical danger?
   ┌──── YES ────► Contact clinic / nearest vet IMMEDIATELY.
   │               Then call client if not present.
   │               Then call MedPup 24‑hr line.
   │               Document everything.
   │
   └──── NO ─────► Can you resolve it yourself in 5 minutes?
                      ┌─ YES → Resolve. Log it.
                      └─ NO  → Call MedPup 24‑hr line for decision support.
                                Then communicate with client.
```
