# Pilot Zone

## Default Pilot Zone

Default assumption for the first pilot:

- Primary area: Largo, Florida
- County: Pinellas County
- Initial service radius: 5–10 ZIP codes around Largo
- Expansion: only after resource coverage and partner contacts are verified

If the project owner chooses a different location, replace this document before launch.

## Pilot-Zone Rule

Do not launch statewide. Do not market as covering all of Florida during Phase 0 or Phase 1.

Recommended public wording:

> We are currently piloting in a limited local service area. If you are outside the pilot area, we may still be able to suggest general resources, but we cannot guarantee local coordination.

## Jurisdiction Questions to Answer

Before launch, identify:

1. Which animal control agency covers each ZIP?
2. Which public shelter receives found/stray animals from each area?
3. What is the local process for found dogs?
4. What is the local process for found cats?
5. What is the local process for injured stray animals?
6. What is the local process for wildlife?
7. Which emergency veterinary hospitals are closest?
8. Which clinics accept urgent same-day cases?
9. Which organizations handle community cats/TNR?
10. Which resources help low-income owners?
11. Which resources help seniors or disabled owners?
12. Which resources help with pet food/supplies?
13. Which partners accept sponsored-care invoices later?

## Pilot Service Limits

During the first pilot:

- Maximum open cases at one time: 5
- Maximum cases in first real-case week: 3–5
- Maximum cases in week 4: 5–10 if workflow is stable
- Animal housing: none
- Public donation solicitation: none unless compliance is complete
- Medical services: none
- Unsupervised volunteers: none
- Transport: only to confirmed destinations with consent and safe containment

## Expansion Criteria

Expand beyond the initial ZIPs only when:

- at least 20 local resources are verified,
- at least 5 partner conversations have occurred,
- emergency clinics are mapped,
- animal control/shelter routing is known,
- the case board works,
- test cases have been completed,
- and the founder can handle the current case volume without unsafe delay.
