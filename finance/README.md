# MedPup Finance

Financial tracking and projection tools for MedPup operations.

## Files

| File | Purpose | Usage |
|------|---------|-------|
| `trip-ledger.csv` | P&L by trip — example data showing ramp to profitability | Import to Google Sheets as base tab |
| `trip-ledger-template.csv` | Blank template with headers | Copy for each new trip entry |
| `projections-scenarios.csv` | 4 growth scenarios (conservative → scale) | Strategic planning |
| `vet-outreach-tracker.csv` | Referral partner pipeline tracker | Import to Airtable/Google Sheets |
| `procedure-cost-comparison.csv` | 16-procedure cost comparison matrix | Website pricing + marketing materials |

## Dashboard Setup

1. Create a new Google Sheet: **MedPup Financial Dashboard**
2. Create tabs: Trip Ledger, Monthly Summary, Concierge Fee Tracker, Marketing Spend, Projection vs Actual
3. Import relevant CSVs into each tab
4. Set up QUERY formulas for Monthly Summary auto-aggregation

## Key Metrics to Track

| Metric | Target | Frequency |
|--------|--------|-----------|
| Trip margin | >25% | Per trip |
| Customer Acquisition Cost (CAC) | <$150 | Monthly |
| Average pets per trip | >4 | Per trip |
| Concierge fee per pet | $285 (Freeport), $1,500 (Cancun) | Per trip |
| Net Promoter Score (NPS) | >70 | Post-trip survey |
